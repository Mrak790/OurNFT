# OurNFT

Setup on windows (once after cloning):

```python
python setup.py
```

Setup on Ubuntu:
```
bash  script.sh
```

Run server (windows):

```python
python runserver.py
```

Run server (ubuntu):

```python
python3 runserver.py
```

/restore - to restore image with secret key
/images/<public token>  - get info about image
